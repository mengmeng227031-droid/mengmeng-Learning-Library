# 口算题库目录

这里预留给 1-6 年级口算题库文件。

建议后续按年级拆分：

- `grade-1.json`
- `grade-2.json`
- `grade-3.json`
- `grade-4.json`
- `grade-5.json`
- `grade-6.json`

当前页面先使用 `oral-math.js` 内置规则生成题目。题库发来后，再把这里的 JSON 接入口算出题器。

题库内容按「河北冀教版 + 新课标（2022）」口算水平组织：

- 一年级：10以内加减、20以内进位加/退位减、整十数加减、两位数±一位数不进退位。
- 二年级：100以内进退位加减、表内乘除法、简单乘加乘减。
- 三年级：万以内整百整千加减、整十整百乘除、两位数乘除一位数、简单同分母分数加减。
- 四年级：整万整亿数加减、两位数乘整十/整百、几百几十乘除、小数简单加减。
- 五年级：小数乘除、异分母分数加减、简单分数乘除、百分数互化。
- 六年级：分数/小数/百分数混合、比的化简和求比值、两步四则混合。
