# mengmengteachv1

这是蒙蒙学习库下一阶段的独立设计目录。当前只整理管理端，不改动现有项目源码。

## 本阶段范围

管理端负责：

- 上传资料
- 解析资料
- 编辑结构化内容
- 分类资料
- 发布资料

用户端暂不在本目录实现。
