# 蒙蒙学习库

蒙蒙学习库是一个面向儿童学习场景的 Windows 客户端 / Web MVP 原型，当前以静态 HTML/CSS/JS 为主，配套一个本地 Node.js 后端用于资料库文件管理。

## 当前功能

- 首页学习入口与品牌化界面
- 口算出题器
- 资料库：支持三级资料目录、文件展示与上传接口
- 好句查询入口
- 英文学习卡片与三年级上册词汇配图
- 音标练习卡片批量生成结果

## 本地使用

直接打开：

```text
index.html
```

如需使用资料库上传和文件接口，启动后端：

```powershell
node back/server.js
```

默认资料目录位于：

```text
back/library
```

## 目录说明

- `index.html`、`styles.css`、`script.js`：客户端主页
- `back/`：本地资料库后端
- `assets/`：品牌、课程和单词配图资源
- `data/`：离线学习数据
- `音标练习/`：音标练习卡片输出
- `scripts/`：数据与图片生成辅助脚本

## 开发原则

项目最终会支持学生账号登录、按账号动态填充学习内容，并接入数据库。后续新增功能应尽量保留用户身份、API 数据源、数据库字段和权限边界的扩展空间。
